Readme for data sets

In this folder are the final data sets used in the models and algorithms used.

THe original raw data is not provided, as there was a lot of aggregation and transformations used to 
prepare the final data sets.  However the links for the original data are provided below. 

- US Prescription rate by County
https://www.cdc.gov/drugoverdose/maps/rxrate-maps.html

- State Policies and Regulatory Policies related to Pain Care 
http://lawatlas.org/datasets/state-laws-and-other-regulatory-policies-related-to-pain-care-final

- Supply chain data (Washington Post DEA Pain Pill Database)
https://www.washingtonpost.com/national/2019/07/18/how-download-use-dea-pain-pills-database/?arc404=true

- CDC Opioid Overdose by county (CDC Wonder Multiple Causes of Death Database)
https://wonder.cdc.gov/mcd-icd10.html

- State policies for Overdoses and Nalaxone access
https://www.networkforphl.org/_asset/qz5pvn/network-naloxone-10-4.pdf
